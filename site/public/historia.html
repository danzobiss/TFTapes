História do TFT

▸ O que é o TFT?

TFT, ou Teamfight Tactics, é um jogo de estratégia no estilo "Auto Chess" que se passa no universo de League of Legends. O objetivo do jogo é ser o último jogador restante após batalhas automáticas que acontecem em cada rodada.

Para avançar nas colocações e sobreviver até o final você precisa ganhar as batalhas de cada rodada, para isso é necessário comprar campeões na loja e posicioná-los em seu tabuleiro. Todo campeão possui sinergias separadas entre classes e origens, nas quais quando campeões diferentes compartilham a mesma sinergia, é ativado um bônus para seu time.

▸ Set 1 - Beta Set

Este foi o primeiro Set do TFT servindo como porta de entrada para esta nova empreitada da Riot Games, mas vamos admitir que não era lá um Set muito bom. As composições eram quebradas, o fator RNG era muito alto, e os gráficos do jogo eram bem estáticos.

Como disse mais cedo, as composições deste Set eram quebradas, então uma composição bem marcante foi uma do finalzinho do Set que era a de Guardiões com Pantheon. Nesta composição, o Pantheom recebia muita Resistência Mágica e Armadura por conta de suas sinergias e se tornava praticamente imortal, ganhando o combate pelo tempo.

▸ Set 2 - Ascensão dos Elementos

Comparando com o Set anterior, este Set já foi melhor. Ele trouxe a temática de elementos junto de mecânicas novas de jogo, sendo as casas elementais a principal. Estas casas elementais forneciam bônus no início da rodada para o campeão que estivesse em cima dela.

Alguns campeões eram muito divertidos de se jogar, por esse motivo nasceu a composição de Zed. Este campeão criava clones, esses clones eram capazes de criar outros clones, e assim por diante; algo que acontecia algumas vezes era colocar o item de feiticeiro no Zed e ele se tornava capaz de criar 2 clones ao invés de 1 só.

▸ Set 3 - Galáxias

Trazendo uma temática futurista e de espaço sideral, neste Set o jogador foi apresentado à mecânica de jogo chamada de Galáxias. Essas galáxias eram variações na partida que deixavam cada uma delas única e não tão monótonas e repetitivas, podendo fazer, por exemplo, com que suas Pequenas Lendas ficassem minúsculas e, consequentemente, com menos vida.

Uma composição marcante deste Set foi a de Cibernéticos, nela um Lucian carregando um Buff Vermelho junto com a sinergia de Pistoleiros queimava o time inimigo inteiro impedindo-os de se curar enquanto o seu time fortalecido finalizava o trabalho.

▸ Set 3.5 - Galáxias: Retorno às Estrelas

Pela primeira vez, houve uma mudança de Set intermediária que trazia algumas mudanças sem alterar o Set completamente. Esse Set intermediário deu tão certo que nos Sets seguintes também houveram essa mudança. Até hoje é considerado o Set mais balanceado que já esteve presente no jogo por conta de campeões que não definiam partidas.

A melhor composição presente aqui foi a Master Yi e Mestres da Lâmina, pois o Master Yi causava Dano Verdadeiro a cada ataque e a sinergia de Mestres da Lâmina faziam com que ele batesse muito rápido, sendo um counter direto da composição de Cassiopeia.

▸ Set 4 - Destinos

O tema deste Set é espíritos e divindades, e traz a mecânica de escolhidos. Tinha tudo pra dar certo, mas a maneira como foi planejada e executada não o favoreceu muito. Essa mecânica de escolhidos fazia aparecer na loja um campeão com status aprimorados e uma de suas sinergias contando como 2, porém o aparecimento deste campeão dependia muito do RNG e quem tivesse o escolhido melhor ganhava o jogo.

A composição mais fácil de fazer foi a Duelistas com Yasuo Carry, pois os duelistas garantiam Velocidade de Ataque e o Yasuo fazia praticamente 1x9 se seu irmão Yone estivesse no time.

▸ Set 4.5 - Destinos: Festival das Feras

Assim como no Set 3.5, houve uma mudança intermediária no Set 4. Esta mudança trouxe uma espécie de "lanterna" pro jogo que dava um item único e aleatório para todos os jogadores, e sinergias mais desbalanceadas ainda. O jogo continuou tóxico por conta da mecânica de escolhidos que permaneceria até o final do Set, e para muitos jogadores foi o pior Set já feito.

Se o jogo já estava tóxico antes, com a composição de Aurelion Sol ficou mais ainda. Nessa composição bastava o Aurelion ter uma Pistola Hextech e uma Manopla Adornada para apagar o time inimigo completamente.

▸ Set 5 - Contestação

Por fim, este é o Set presente no jogo nos dias de hoje que traz a temática de Bem vs. Mal. A mais nova mecânica presente são os itens sombrios que são os mesmos itens antigos mas fortalecidos pela Névoa Negra, porém com o preço de redução de status do campeão que carrega este item.

A composição que vem sendo bastante utilizada nas filas ranqueadas é a de LeBlanc com Buff Azul Sombrio, pois o Buff Azul Sombrio recupera 50 de mana do usuário toda vez que usa sua habilidade, isso faz com que a LeBlanc use sua habilidade repetidas vezes atordoando o time inimigo para sempre.
